String Methods And Properties

Before starting string methods and properties, we need to know that strings are indexed. So basically we can access any single , character of a string by doing like this.

Expression: 'Hello World'[1]

Result:'e'

There is only one property that exists

Property

  • length

Methods (without Argument)

  • toLowerCase()
  • trim()
  • trimStart()
  • trimEnd()

Methods (with Argument)

  • indexof()
  • replace()
